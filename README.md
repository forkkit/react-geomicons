# react-geomicons

## Getting Started

```bash
npm install --save-dev react-geomicons
```

```js
var React = require('react');
var Geomicon = require('react-geomicons');

module.exports = React.createClass({
  render: function() {
    <MyComponent>
      <Geomicon icon="bookmark" />
    </MyComponent>
  }
});
```
